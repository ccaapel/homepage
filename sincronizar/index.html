Sincronizador Sponte

Raspa os dados de alunos do Sponte e envia para a Base de Alunos do painel CCAA.

1
Cole a URL do backend de alunos (o App da Web do projeto ALUNOS CCAA, termina em /exec).
Fica salva neste navegador — a mesma usada no painel.
2
Arraste o botão abaixo para a barra de favoritos do seu navegador.
Faça isso uma vez só — fica salvo para sempre.

🔄 Sincronizar CCAA
3
Abra o Sponte e acesse Cadastros → Alunos.
Certifique-se de que a lista está mostrando os alunos Ativos.
4
Clique no favorito "Sincronizar CCAA" que você acabou de criar.
Uma barra de progresso aparecerá no topo da página do Sponte.
O processo navega pelas páginas automaticamente e leva cerca de 1–2 minutos para 388 alunos.
5
Quando aparecer "✅ Sincronização concluída!", volte ao painel e abra Base de Alunos.
Clique em Recarregar se os alunos não aparecerem imediatamente.
⚠️ Aniversários: o Sponte não exibe a data de nascimento na lista geral. O sincronizador visitará automaticamente o perfil de cada aluno para capturar essa data — por isso demora um pouco mais. Se o processo travar, clique novamente no favorito para retomar de onde parou.
